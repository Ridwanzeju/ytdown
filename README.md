# 🎬 Free Download Video YT — Web By BANG RZ

Download video YouTube gratis, tanpa iklan, dengan backend Node.js.

---

## 📁 Struktur Folder

```
ytdown/
├── server.js        ← Backend Express
├── package.json     ← Daftar dependencies
└── public/
    └── index.html   ← Tampilan web
```

---

## 🚀 Cara Menjalankan

### 1. Install Node.js
Download di: https://nodejs.org (pilih versi LTS)

### 2. Install dependencies
Buka terminal/cmd di folder `ytdown`, lalu ketik:
```bash
npm install
```

### 3. Jalankan server
```bash
npm start
```

### 4. Buka di browser
```
http://localhost:3000
```

---

## 🌐 Deploy ke Internet (Gratis)

### Pakai Railway (Paling mudah):
1. Daftar di https://railway.app
2. Klik **New Project → Deploy from GitHub**
3. Upload folder ini ke GitHub dulu
4. Railway otomatis detect Node.js dan deploy!
5. Dapat link publik gratis ✅

### Pakai Render:
1. Daftar di https://render.com
2. New → Web Service → Upload repo GitHub
3. Build Command: `npm install`
4. Start Command: `npm start`
